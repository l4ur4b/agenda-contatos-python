# agenda-contatos-python
📒 Agenda de Contatos em Python

Uma agenda de contatos simples desenvolvida em Python, com o objetivo de praticar conceitos básicos da linguagem e lógica de programação.

O programa roda no terminal e permite gerenciar contatos através de um menu interativo.

🚀 Funcionalidades

Adicionar contatos

Visualizar todos os contatos

Atualizar informações de um contato

Deletar contatos

Favoritar ou desfavoritar contatos ⭐

Visualizar apenas contatos favoritados

🧠 Conceitos praticados

Este projeto utiliza alguns conceitos fundamentais de Python:

Funções

Listas

Dicionários

Estruturas condicionais (if / elif)

Estruturas de repetição (while)

enumerate()

List comprehension

Manipulação de dados no terminal

💻 Tecnologias utilizadas

Python 3
